### INFORMACIÓN DEL PROYECTO

Título: Stranger Things - Catálogo de Personajes
Descripción: Sistema web de gestión CRUD de personajes de la serie Stranger Things, 
desarrollado con PHP, MySQL, JavaScript y AJAX.

Repositorio: https://github.com/usuario/final-p2-acn2bv-lukmanov

### CARACTERÍSTICAS IMPLEMENTADAS

✅ Base de datos MySQL con 12 personajes iniciales
✅ API REST con endpoints JSON
✅ CRUD completo (Crear, Leer, Actualizar, Eliminar)
✅ Búsqueda por nombre en tiempo real
✅ Filtro por rol (Niños, Adolescentes, Adultos, Villanos)
✅ Paginación (5 personajes por página)
✅ AJAX con Fetch API (sin recargar página)
✅ Temas claro/oscuro con sesiones PHP
✅ Diseño responsive (mobile, tablet, desktop)
✅ Validaciones frontend (HTML5) y backend (PHP)
✅ SweetAlert2 para notificaciones
✅ Animaciones y efectos visuales
✅ Código limpio y documentado

### TECNOLOGÍAS UTILIZADAS

- Backend: PHP 7.4+
- Base de Datos: MySQL 5.7+
- Frontend: HTML5, CSS3, JavaScript ES6+
- AJAX: Fetch API
- Biblioteca: SweetAlert2
- Control de versiones: Git
